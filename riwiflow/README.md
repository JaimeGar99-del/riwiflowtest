# RIWIFLOW

Web application developed with **Vite**, **Vanilla JavaScript**, **TailwindCSS**, and **JSON Server**, focused on delivering a modern, fast, and dynamic experience.

---

# Description

RIWIFLOW is a frontend project built with a modular architecture using pure JavaScript and modern development tools.

The project uses:

- Vite as the development environment
- TailwindCSS for styling
- JSON Server to simulate a REST API
- Modular architecture in JavaScript

---

# Technologies Used

- JavaScript ESModules
- Vite
- TailwindCSS
- JSON Server
- HTML5
- CSS3

# Project Installation

## Clone the repository

```bash
git clone <REPOSITORY_URL>
```

---

## Enter the project folder

```bash
cd riwiflow
```

---

## Install dependencies

```bash
npm install
```

---

# Run the Project

## Start Vite

```bash
npm run dev
```

---

## Start JSON Server

```bash
npm run server
```

The API will run at:

```bash
http://localhost:3000
```

The frontend application will usually run at:

```bash
http://localhost:5173
```
